# Exploding Kittens

A web app to play the Exploding Kittens card game with friends. Pass-and-play mode for 2–10 players.

## Play

[Play the game](https://yukkta-seelam.github.io/Exploding-Kittens/) (after enabling GitHub Pages)

## Setup

1. Clone this repository
2. Enable GitHub Pages: Settings → Pages → Source: GitHub Actions
3. Push to trigger deployment

## Game Modes

- **Base Game**: 2–5 players
- **Party Pack**: 2–10 players

## How to Play

Pass the device between players. On your turn, play any cards (or none), then draw a card. Avoid drawing Exploding Kittens—or use a Defuse to survive!
